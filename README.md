# E-Commerce Data Analysis Project

Analysis of customer behavior, product preferences, and transaction patterns using e-commerce dataset. Includes EDA, lookalike modeling, and customer segmentation.

## Dataset
- `Customers.csv`: Customer profiles and signup information
- `Products.csv`: Product catalog with categories and pricing
- `Transactions.csv`: Transaction records with purchase details

## Project Structure
- `FirstName_LastName_EDA.ipynb`: Exploratory Data Analysis notebook
- `FirstName_LastName_EDA.pdf`: Business insights report
- `FirstName_LastName_Lookalike.ipynb`: Customer similarity model implementation
- `FirstName_LastName_Lookalike.csv`: Top 3 similar customers for IDs C0001-C0020
- `FirstName_LastName_Clustering.ipynb`: Customer segmentation analysis
- `FirstName_LastName_Clustering.pdf`: Clustering results and metrics

## Requirements
- Python 3.8+
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Tasks
1. **EDA**: Analysis of customer behavior and transaction patterns
2. **Lookalike Model**: Customer similarity recommendations
3. **Customer Segmentation**: Clustering analysis with DB Index evaluation
